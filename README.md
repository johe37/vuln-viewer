# Vulnerability Viewer (Trivy)

A simple static web UI for browsing Trivy JSON reports.

---

## 📦 Features

- Browse Trivy rootfs scan reports
- Search vulnerabilities by package or CVE
- View severity summary
- Lightweight (no frameworks)
- Works offline

---

# Start

```
# generate reports
./scripts/generate-report.sh

# start server
./scripts/start.sh
```
